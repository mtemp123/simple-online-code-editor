# simple-online-code-editor
Simple code editor (HTML+CSS+JS &amp; Preview)

* [Bootstrap 4.3.1](https://github.com/twbs/bootstrap)
* [Lobibox 1.2.7.mod](https://github.com/mtemp123/lobibox)
* [Ace 1.4.5](https://github.com/ajaxorg/ace)
* [Font-Awesome 5.10.1](https://github.com/FortAwesome/Font-Awesome)
* [JQuery 3.4.1](https://github.com/jquery/jquery)
